# Coding with Agents: Intro

*Coding with Agents* is a practical book about using coding agents as part of real software work: planning changes, reading unfamiliar repositories, implementing safely, testing, reviewing, and shipping.

The book is written for engineers who want agent-assisted development to feel less like a prompt trick and more like a repeatable working system. It focuses on the operating habits around agents: how to scope tasks, how to keep context clean, how to verify output, and how to decide when human judgment should take over.
