Coding with Agents 导读

Coding with Agents 是一本关于如何把编程智能体放进真实软件开发流程里的实践书。它关注的不是单次提示词技巧，而是从理解代码库、拆解任务、实现改动、验证测试、审查结果到发布交付的一整套工作方式。

这本书面向已经会写代码、但希望更稳定地使用智能体协作的工程师。核心问题是：怎样给智能体足够清晰的上下文，怎样限制变更范围，怎样验证输出，怎样在人类判断和自动化之间划清边界。

这篇导读只是入口。完整书稿在这里：

阅读 Coding with Agents
